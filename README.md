# 🏏 IPL Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze Indian Premier League (IPL) matches from 2008–2025. It provides insights into team performance, player statistics, venue analysis, season trends, and match outcomes.

---

## 🚀 Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV
- GitHub

---

## 📊 Dashboard Pages

### 1️⃣ IPL Overview
- Total Matches
- Total Teams
- Total Venues
- Total Seasons
- Top Winning Teams
- Matches by Venue
- Matches Played by Season
- Toss Decision Analysis

### 2️⃣ Team Performance

- Total Wins

- Wins Across Seasons

### 3️⃣ Player Performance

- Average Strike Rate
- Average Economy

- Top 10 Wicket Takers

### 4️⃣ Match Insights
- Toss Decision Analysis
- Toss Winner Conversion
- Matches by Venue
- IPL Growth Trend
