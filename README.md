# 🏏 IPL Analysis Dashboard

An interactive Power BI dashboard that analyzes Indian Premier League (IPL) matches from 2008–2025. This project provides insights into team performance, player statistics, venue trends, season growth, and match outcomes using Power BI, DAX, and Power Query.

---

# 📌 Project Overview

The objective of this project is to transform raw IPL match data into meaningful business insights through interactive dashboards. The dashboard enables users to explore historical IPL data using dynamic visualizations, KPIs, filters, and navigation across multiple report pages.

This project demonstrates practical data analytics skills including data cleaning, transformation, data modeling, DAX calculations, dashboard design, and business storytelling.

# 📈 Key Business Insights

- Mumbai Indians and Chennai Super Kings are among the most successful IPL teams.
- Winning the toss provides an advantage in many matches but does not guarantee victory.
- Certain venues consistently host a significantly higher number of IPL matches.
- IPL has experienced steady growth in the number of matches played across seasons.
- Player performance can be analyzed using multiple metrics such as runs, strike rate, wickets, and economy rate.
